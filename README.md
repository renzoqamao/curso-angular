# Primeros pasos

## Requisitos

- [nvm](https://github.com/nvm-sh/nvm)

    ```bash
    nvm current # muestra la version de node actual
    nvm list # muestra las versiones de node instaladas
    nvm list available # muestra las versiones de nodes que pueden ser instaladas
    nvm use <version> # cambia a la version de node especificada
    nvm install <version> # instala la version de node especificada
    ```
- [nodeJs](https://nodejs.org/en/download/current)

    > nota: no es necesario nvm, se puede descargar directamente NodeJS.

- Angular: una vez instalada nodeJS, podemos instalar angular.

    ```bash
    npm install -g @angular/cli # instalación global de angular
    ```
